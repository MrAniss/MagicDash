## Conseils

- **Format** : PNG (JPEG accepté mais moins net pour du texte)
- **Largeur** : 1600px ou 1920px (le viewport max du dashboard est 1600px)
- **Anonymisation** : si tu publies sur un repo public, **floute ou maquille** les noms de marques, customer IDs, montants sensibles. Outils gratuits : [Greenshot](https://getgreenshot.org/), [ShareX](https://getsharex.com/), ou simplement Photoshop/GIMP.
- **Mode démo** : tu peux temporairement modifier `accounts.js` avec des noms factices (`Marque A`, `Marque B`) le temps des screenshots, puis revert.
- **Compression** : avant commit, passe les PNG dans [tinypng.com](https://tinypng.com) — divise la taille par 3-5 sans perte visible.

## Sur Windows

Raccourci natif : **Win + Shift + S** → sélectionne la zone → coller dans Paint → sauvegarder en PNG.

Ou installe **ShareX** pour des captures avec délai, watermark, upload auto, etc.

## Une fois ajoutés

Les images apparaîtront automatiquement dans le README rendu sur GitHub. Pas de config supplémentaire.
